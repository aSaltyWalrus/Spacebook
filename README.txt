JavaScript style guide used: AirBnB (https://github.com/airbnb/javascript)

Spacebook incorporates the fundamental functionalities of a social media mobile application. This encompasses features such as:
- Account creation and login, with session tokens implemented
- Account editing (password and username changes)
- Friend management (request/adding/deleting)
- Profile pages with customisation such as a profile picture and user information
- Ability to post on profile page as well as onto other user's pages
- Searching functionality to find and connect to other users using SQL, with the ability to limit search results
- CSS styling
 
NOTE: this project was an exercise in front-end development. API available at: https://github.com/ash-williams/Spacebook
